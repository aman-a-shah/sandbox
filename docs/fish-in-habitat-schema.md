# Fish-In-Habitat JSON Schema

This document describes the JSON generated by:

```bash
node scripts/build-fish-per-habitat.js "habitat query"
```

## Top-Level Shape

```json
{
  "habitat": {},
  "notes": {},
  "fishCount": 0,
  "fish": []
}
```

## `habitat`

```json
{
  "habitatId": "string",
  "habitatInformationName": "string",
  "jnccCode2015": "string",
  "jnccName2015": "string",
  "jnccCode2022": "string",
  "jnccName2022": "string",
  "eunisCode2008": "string",
  "eunisName2008": "string",
  "eunisCode2022": "string",
  "eunisName2022": "string",
  "reviewDate": "string",
  "url": "string",
  "inferredDepthZone": "string",
  "inferredSubstratum": ["string"]
}
```

## `notes`

```json
{
  "likelyCatchPercent": "string",
  "observedSpeciesRecordPercent": "string",
  "fishingActivityAssociatedPercent": "string",
  "sourceApis": ["string"]
}
```

## `fishCount`

`number`

The number of fish entries included in the `fish` array.

## `fish`

Each entry in `fish` has this shape:

```json
{
  "commonName": "string",
  "scientificName": "string",
  "family": "string",
  "order": "string",
  "aphiaId": "string|null",
  "marlinSpeciesId": "string|null",
  "marlinUrl": "string|null",
  "fishBaseSummaryUrl": "string|null",
  "fishBaseEnvironment": "string|null",
  "fishBaseEcology": "string|null",
  "sizeCategory": "string|null",
  "averageDepthMeters": "number|null",
  "fishBaseCommonLengthCm": "number|null",
  "wormsDistributionCount": "number",
  "wormsDistributionSummary": ["string"],
  "habitatFitPercent": "number",
  "likelyCatchPercent": "number",
  "observedSpeciesRecordPercent": "number",
  "fishingActivityAssociatedPercent": "number"
}
```

## Field Notes

- `habitatFitPercent` is an internal suitability score used to rank fish against the selected habitat.
- `likelyCatchPercent` is the best game-facing field for catch weighting.
- `observedSpeciesRecordPercent` is the best field for observation-style weighting.
- `fishingActivityAssociatedPercent` is a proxy for fishery association, not a direct species-level Global Fishing Watch measurement.
- `averageDepthMeters` is derived from FishBase summary depth text when available.
