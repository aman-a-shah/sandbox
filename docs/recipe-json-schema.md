# Recipe JSON Schema

This document describes the JSON generated by:

```bash
node scripts/build-recipes-for-fish.js "<fish-by-habitat-json>" "<fish name>"
```

## Top-Level Shape

```json
{
  "generatedAt": "string",
  "fishRecipeCount": 0,
  "fishRecipes": []
}
```

## `generatedAt`

`string`

ISO timestamp for the latest update to the master recipe file.

## `fishRecipeCount`

`number`

The number of fish entries currently stored in the master recipe file.

## `fishRecipes`

Each item in `fishRecipes` has this shape:

```json
{
  "fish": {},
  "notes": {},
  "recipeCount": 0,
  "recipes": []
}
```

```json
{
  "commonName": "string",

  "observedSpeciesRecordPercent": "number",
  "fishingActivityAssociatedPercent": "number",
  "sizeCategory": "string|null",
  "averageDepthMeters": "number|null"
}
```

## `fishRecipes[].notes`

```json
{
  "quotaStrategy": "string",
  "filtering": "string",
  "pricing": "string",
  "quantityInference": "string",
  "fallback": "string"
}
```

## `fishRecipes[].recipeCount`

`number`

The total number of recipes currently stored for that specific fish.
Each run attempts to add at most `1` new recipe for the selected fish, and appends it if it is not already present.

## `fishRecipes[].recipes`

Each recipe entry has this shape:

```json
{
  "requiredFishIngredients": [],
  "recipeId": "number",
  "title": "string",
  "sourceUrl": "string|null",
  "image": "string|null",
  "readyInMinutes": "number|null",
  "servings": "number|null",
  "popularityScore": "number|null",
  "fishUsed": {},
  "rarity": "string",
  "estimatedPrice": {},
  "nutrition": {},
  "recipe": {}
}
```

## `requiredFishIngredients`

This is a list of all ingredient entries in the recipe that match the selected fish.

```json

]
```

## `recipe.ingredients`

Each ingredient has this shape:

```json
{
  "name": "string|null",
  "amount": "number|null",
  "unit": "string|null",
  "original": "string|null",
  "isRequiredFishIngredient": "boolean"
}
```

## Notes

- `requiredFishIngredients` is the summary list of all fish ingredient entries matching the selected fish.
- `recipe.ingredients[].isRequiredFishIngredient` marks those same ingredient rows inline.
- `quantityWholeFish` is inferred from explicit units when available, otherwise from mass and fish length.
- `rarity` is derived from fish scarcity and recipe fish quantity, not returned directly by Spoonacular.
- The output file is a single master file at `generated/recipes-by-fish/recipes.json`.
- Re-running the script appends a new recipe under the matching fish entry when it finds one with a new `recipeId`.
- If Spoonacular finds no recipes for the selected fish, the script generates one fallback fish recipe tailored to the fish type.
